## About Game

## How to install Project
- Clone project > ``` git clone https://github.com/tiekungsos/cardChallengeProject.git ```
- copy .env-dist > .env
- set up docker > https://docs.docker.com/engine/install/ubuntu/
- Install project > ``` bash install.sh ```
- enjoy the game.

### Stack
- nginx 1.13
- php 7.4
- mariadb:latest
